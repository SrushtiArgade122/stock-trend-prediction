# Stock Trend Prediction

## 📌 Overview
This project predicts stock market trends using historical data from Nifty50. It applies data preprocessing, exploratory data analysis (EDA), and machine learning models to forecast stock prices.

## 🚀 Features
- Data cleaning and preprocessing
- Outlier detection using IQR
- Correlation analysis and visualization
- Linear regression model for prediction
- Performance evaluation (R², MAE, RMSE)

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 📊 Model Performance
- R² Score: ~0.99
- RMSE: ~50

## 📁 Files
- stock-trend-prediction.ipynb
- Nifty50.csv
